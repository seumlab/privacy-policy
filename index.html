개인정보처리방침

(주)세움디지털(이하 ‘회사’)는 이용자의 개인정보를 소중하게 여기며, 「개인정보 보호법」을 비롯한 관련 법령을 준수합니다.

회사는 다음과 같은 개인정보처리방침을 통해, 사용자의 개인정보가 어떤 방식으로 수집·이용·보관되는지 안내드립니다.

1. 수집하는 개인정보 항목

회사는 아래와 같은 개인정보를 수집할 수 있습니다.

  • ① Google 로그인 시
    - 이메일 주소 (Google 캘린더 연동 및 식별 목적)
    - Google Calendar 이벤트 ID (일정 연동 및 중복 방지 목적)
  • ② 앱 기능 이용 시
    - 일정 제목, 내용, 시작일/종료일, 시간
    - 중요도 설정 여부, 알림 설정 정보
  • ③ 날씨 기능 이용 시
    - 위치정보 (기기의 GPS 좌표)
  • ④ 알림 기능 이용 시
    - 일정별 알림 시간 설정 정보 (기기 내부 저장)

※ 위 정보는 사용자가 해당 기능을 사용할 때에만 수집되며, 일부 항목은 선택적으로 수집됩니다.

2. 개인정보 수집 및 이용 목적

  • Google 캘린더 연동을 통한 일정 동기화
  • 사용자 일정 관리 및 알림 제공
  • 날씨 정보를 기반으로 한 일정 표시 기능
  • 사용자 맞춤형 기능 설정 유지

3. 개인정보의 보유 및 이용 기간

- 사용자의 개인정보는 수집·이용 목적이 달성되면 즉시 파기됩니다.
- 사용자가 앱을 삭제하거나 Google 계정 연동을 해제한 경우, 해당 정보도 함께 삭제됩니다.

4. 개인정보의 제3자 제공

회사는 이용자의 개인정보를 외부에 제공하지 않습니다.
단, 다음의 경우는 예외로 합니다:

  • 이용자가 사전에 동의한 경우
  • 법령에 따른 요청이 있는 경우

5. 개인정보 처리 위탁

회사는 개인정보를 외부에 위탁하지 않습니다.

6. 이용자의 권리 및 행사 방법

  • Google 계정 연동 해제
  • 앱 내 일정 삭제 또는 수정
  • 앱 삭제를 통한 정보 삭제

※ 관련 문의는 아래 이메일을 통해 접수받습니다.

7. 개인정보 자동 수집 장치의 설치/운영 및 거부

- 회사는 쿠키 등의 자동 수집 장치를 사용하지 않습니다.
- 단, Google API 사용 시 일부 인증 정보가 자동 처리될 수 있습니다 (Google의 개인정보처리방침에 따름).

8. 개인정보 보호를 위한 기술적·관리적 대책

  • 일정 정보는 사용자의 기기 내 로컬 DB에 저장되며 외부로 전송되지 않습니다.
  • Google 연동 시 발급되는 토큰은 안전하게 저장되며, 타 서비스에 사용되지 않습니다.
  • 알림 기능은 기기 내부 알람 시스템을 통해 처리됩니다.
